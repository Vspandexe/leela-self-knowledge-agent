# Лила — скилл-проводник игры самопознания для ИИ-агента

Скилл превращает твоего Claude-агента в **ведущего игры Лила** (Leela / Джняна-чаупада) — древней индийской настольной практики самопознания на 72 клетки. Агент помогает сформулировать запрос, кидает кубик, ведёт по доске, раскрывает смысл каждой клетки и задаёт вопросы для рефлексии — как живой проводник трансформационной игры.

Механику полностью считает встроенный движок (`leela.py`): броски, правило шестёрок, стрелы и змеи, цель — клетка 68. Агент отвечает за смысл и диалог. **Состояние партии хранится в файле — игра переживает перезапуск агента, можно продолжить в любой момент.**

> Основано на каноне Хариша Джохари («Лила. Игра самопознания», 1980) — с полной, выверенной картой всех 72 клеток, 9 стрел и 10 змей.

## Что это

Лила — не «предсказание», а инструмент рефлексии. Игрок входит в игру с **запросом** (вопрос, тема, ситуация), и клетки, на которые он попадает, отражают то, что актуально в его жизни сейчас. Стрелы — прорывы и добродетели, поднимающие вверх; змеи — непрожитые уроки, откатывающие назад. Цель — прийти к клетке 68 (Космическое Сознание).

## Установка агенту

**Самый простой способ — скинь этот промпт своему агенту, он поставит скилл сам:**

> Установи мне скилл игры Лила из этого репозитория: https://github.com/Vspandexe/leela-self-knowledge-agent
>
> Сделай так:
> 1. Склонируй репозиторий во временную папку (git clone).
> 2. Скопируй папку `skills/leela` в директорию скиллов Claude Code: `~/.claude/skills/` (создай её, если нет). Должно получиться `~/.claude/skills/leela/` с файлами SKILL.md, leela.py, board.json, rules.md, cells.md.
> 3. Проверь, что установлен python3. Запусти для проверки: `python3 ~/.claude/skills/leela/leela.py board 1` — должно вывести описание клетки 1 (Рождение).
> 4. Подтверди, что скилл на месте, и коротко скажи, как начать партию.
>
> После установки я смогу написать «проведи мне сессию Лилы» — и ты станешь ведущим игры.

**Вручную:**

1. Скопируй папку `skills/leela/` в директорию скиллов своего агента (для Claude Code — `~/.claude/skills/leela/`):
   ```bash
   cp -r skills/leela ~/.claude/skills/
   ```
2. Убедись, что установлен `python3` (используется только стандартная библиотека — ничего доставлять не нужно).
3. Готово. Скилл подхватится автоматически по описанию.

Подробнее — [`docs/install-to-agent.md`](docs/install-to-agent.md).

## Как играть

Просто напиши агенту:

> Проведи мне сессию Лилы. Мой запрос — «…».

Агент начнёт партию, попросит уточнить намерение и будет вести игру ход за ходом. Можно играть **цифровым кубиком** (агент кидает сам) или **физическим** (называешь агенту выпавшие числа). Подробнее — [`docs/how-to-play.md`](docs/how-to-play.md).

## Структура

```
skills/leela/
├── SKILL.md      — инструкция агенту: роль ведущего и сценарий сессии
├── leela.py      — движок игры (броски, правило шестёрок, стрелы/змеи, цель, сохранение партии)
├── board.json    — данные доски: 72 клетки, 9 стрел, 10 змей
├── rules.md      — полные правила (канон Джохари)
└── cells.md      — значения всех 72 клеток + вопросы для рефлексии
docs/
├── install-to-agent.md — установка скилла агенту
└── how-to-play.md      — как проходить сессию
```

## Движок напрямую (по желанию)

```bash
cd skills/leela
python3 leela.py new --intention "мой запрос"   # начать партию
python3 leela.py roll                            # ход (цифровой кубик)
python3 leela.py roll --dice 6,4                 # ход физическим кубиком
python3 leela.py status                          # где я сейчас
python3 leela.py board 54                        # описание клетки
```

## О точности

Карта клеток, стрел и змей выверена по канону Хариша Джохари (русское издание). Существуют и другие вариации доски Лилы у разных школ — эта следует классической версии Джохари, наиболее проработанной и распространённой.

## Дисклеймер

Лила — инструмент саморефлексии и самопознания, а не медицинская, психотерапевтическая или гадательная практика. Скилл не заменяет профессиональную помощь. Используй осознанно.
