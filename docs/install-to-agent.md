# Установка скилла Лилы своему агенту

Скилл рассчитан на Claude Code (CLI-агент), но подойдёт любому агенту, который умеет читать скиллы формата `SKILL.md` и запускать `python3`.

## Требования

- `python3` (3.8+). Используется только стандартная библиотека — ничего доустанавливать не нужно.
- Агент с поддержкой скиллов (Claude Code и совместимые).

## Шаги (Claude Code)

1. Склонируй репозиторий или скачай папку `skills/leela`.
2. Скопируй её в директорию скиллов агента:
   ```bash
   cp -r skills/leela ~/.claude/skills/
   ```
   Если у твоего агента скиллы лежат в другом месте (например, папка проекта `.claude/skills/` или отдельная директория флота) — копируй туда.
3. Проверь, что Python на месте:
   ```bash
   python3 ~/.claude/skills/leela/leela.py board 1
   ```
   Должно вывести описание клетки 1 (Рождение). Если вывелось — скилл готов.

Скилл активируется автоматически: когда игрок просит «поиграть в Лилу» / «провести сессию Лилы», агент подхватит `SKILL.md` по описанию.

## Где хранится партия

Состояние игры пишется в файл `~/.leela/session.json` (создаётся при первой партии). Благодаря этому **партия переживает перезапуск агента** — после рестарта просто продолжай.

- Сменить путь: переменная окружения `LEELA_STATE=/путь/session.json` или флаг `--state /путь/session.json` у команд движка.
- Несколько игроков/партий — храни в разных файлах через `--state`.

## Обновление

Просто перекопируй свежую папку `skills/leela` поверх старой. Данные доски (`board.json`) и правила при этом обновятся; активные партии в `~/.leela/session.json` не затрагиваются.

## Проверка после установки

```bash
cd ~/.claude/skills/leela
python3 leela.py new --intention "проверка"
python3 leela.py roll --dice 6,4     # вход в игру
python3 leela.py status
python3 leela.py reset               # убрать тестовую партию
```
